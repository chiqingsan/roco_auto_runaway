洛克王国自动避战
使用autohotkey制作, 纯按键模拟, 无内存读取

**自动识别图色, 自动进行避战操作, 功能如下:**

1. 自动聚能: 进入战斗后自动进行聚能操作, 在场上精灵死亡之后自动逃跑
2. 自动逃跑: 进入战斗之后自动逃跑
3. 自动牵手: 检测到牵手选项后自动进行牵手操作(因为我改键了, 所以按键输入的是e键, 默认是f键)

```
如果需要把e改成f, 可以在源码中搜索, 把:
SendKey("e")
改成
SendKey("f")
即可
```

主要牵手小号的时候给小号使用的, 进战后小号可以自动逃跑
牵小号刷污染的时候, 小号可以自动聚气
如果不小心牵手断了, 也可以自动牵手


测试设备为**2560x1440全屏, 暂不支持窗口化游戏(后续计划支持)**   
最大可能进行了不同分辨率的兼容, 理论上所有分辨率都可使用
